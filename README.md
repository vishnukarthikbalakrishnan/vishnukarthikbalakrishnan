# Vishnu Karthik Balakrishnan

M.Sc. Intelligent Manufacturing student at TU Clausthal, working on bearing diagnostics and remaining useful life prediction using vibration data and machine learning.

My master’s thesis is in collaboration with Piller Blowers & Compressors GmbH. It includes a benchmark across public bearing datasets such as NASA IMS and XJTU-SY, and an anomaly detection case study on a real industrial fan. The code and methodology are here: [bearing-rul-prediction](https://github.com/vishnukarthikbalakrishnan/bearing-rul-prediction)

My background is in mechanical engineering, so I usually start from the physics: envelope analysis, bearing fault frequencies, kurtosis, and failure modes. Then I use learned features where they add value. I care a lot about honest validation, including results that did not work as expected.

I also worked on a PLC tank level control project in Structured Text: [PLC_Tank_Level_PID](https://github.com/vishnukarthikbalakrishnan/PLC_Tank_Level_PID)

I’m currently finishing my degree and looking for roles in predictive maintenance, condition monitoring, or rotating equipment reliability in Germany.
